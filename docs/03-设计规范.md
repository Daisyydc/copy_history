## 2. 色彩系统

### 主色调

| 名称 | HEX | RGB | 用途 |
|------|-----|-----|------|
| 芭蕾粉 (Ballet Pink) | `#F2C4CE` | 242, 196, 206 | 渐变、品牌色 |
| 浅粉 (Light Pink) | `#FBE9ED` | 251, 233, 237 | 搜索栏背景、图标底色 |
| 深粉 (Deep Pink) | `#E8A0B4` | 232, 160, 180 | 强调色 |
| 玫瑰粉 (Rose Pink) | `#D4788F` | 212, 120, 143 | 文字标题、图标、爱心主色 |

| 名称 | HEX | 用途 |
|------|-----|------|
| 暖白 (Warm White) | `#FFFAFB` | 面板背景 |
| 纯白 | `#FFFFFF` | 卡片背景 |
| 系统灰 | 系统自动 | 时间戳、placeholder |

### 功能色

| 名称 | HEX | 用途 |
|------|-----|------|

## 3. 应用图标

### 设计
- **白色圆形底**：纯净、柔和
- **玫瑰粉爱心**：`#D4788F`，居中偏上
- **芭蕾粉翅膀**：`#F2C4CE`，爱心两侧
- **抗锯齿边缘**：SDF + smoothstep 柔化过渡
- **适用尺寸**：16~1024px

### 图标规格

| 尺寸 | 用途 |
|------|------|
| 16×16 / 32×32 | 菜单栏小图标 |
| 128×128 | Finder 小图标 |
| 256×256 | Finder 中图标 |
| 512×512 | Finder 大图标 / Dock |
| 1024×1024 | App Store |

## 6. 组件

### 6.1 顶部品牌栏

```
┌──────────────────────────────────────┐
│ 🩰  Ballet Clipboard         N 条    │
│ ░░░░░░░░░░ 粉色渐变背景 ░░░░░░░░░░░░ │
└──────────────────────────────────────┘
```

### 6.2 卡片 (ClipCardView)

```
┌──────────────────────────────────────┐
│ ┌────┐                               │
│ │ 🖼️ │  预览文字内容...         📋  │
│ └────┘  刚刚                         │
│ ← 金色边框（置顶时）→               │
└──────────────────────────────────────┘
```

- 类型图标：浅粉底 + 对应 SF Symbol
- 文字卡片：显示内容前 2 行
- 图片卡片："🖼️ 图片"
- 链接卡片："🔗 域名"
- 置顶：金色描边 `#D4A853`
- 右侧：📋 复制图标提示

### 6.3 搜索栏 (SearchBarView)

```
┌──────────────────────────────────────┐
│ 🔍  搜索复制过的内容...         ✕   │
│ └ 浅粉底 + 芭蕾粉边框 ┘             │
└──────────────────────────────────────┘
```

### 6.4 空状态 (EmptyStateView)

```
┌──────────────────────────────────────┐
│                                      │
│              🩰                      │
│       等待你的第一次复制             │
│   试试在任意 App 中按 ⌘C            │
│   复制的文字会出现在这里 ✨          │
│                                      │
└──────────────────────────────────────┘
```
