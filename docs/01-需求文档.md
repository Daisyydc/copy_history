# 01 — 需求文档

> 项目：Ballet Clipboard（芭蕾剪贴板）
> 版本：1.0
> 最后更新：2026-06-17

---

## 1. 项目概述

面向 macOS 用户的粉色芭蕾风格剪贴板历史管理工具。常驻菜单栏，自动记录用户复制的文字、图片、链接，支持搜索、置顶、删除，可按设定期限自动清理历史。

---

## 2. 用户画像

- macOS 日常用户
- 经常复制粘贴文字、图片、链接
- 需要找回之前复制过的内容
- 喜欢可爱、少女风格的界面设计

---

## 3. 功能清单（优先级排序）

### P0 — 核心功能（必须有）

| 编号 | 功能 | 描述 |
|------|------|------|
| F01 | 剪贴板监听 | 自动捕获 Cmd+C 复制的内容（文字、图片、链接） |
| F02 | 历史记录存储 | 文字、图片原图、链接全部存入本地数据库 |
| F03 | 菜单栏图标 | 顶部菜单栏常驻芭蕾图标，点击弹出面板 |
| F04 | 历史列表展示 | 时间降序排列，卡片式展示每一条复制记录 |
| F05 | 点击复制回剪贴板 | 点击卡片 → 内容回到剪贴板 → 用户 Cmd+V 粘贴 |

### P1 — 重要功能

| 编号 | 功能 | 描述 |
|------|------|------|
| F06 | 搜索 | 顶部搜索栏，实时过滤文字/链接内容 |
| F07 | 置顶 | 重要卡片可置顶，始终排在最上方 |
| F08 | 删除 | 单条删除历史记录 |
| F09 | 保留期限设置 | 可选 1天 / 3天 / 5天，过期自动清理 |

### P2 — 体验增强

| 编号 | 功能 | 描述 |
|------|------|------|
| F10 | 开机自启动 | 电脑开机后自动在后台运行 |
| F11 | 深色/浅色模式 | 适配 macOS 外观设置 |
| F12 | 复制反馈 | 点击卡片后显示"已复制"提示 |
| F13 | 动画效果 | 卡片进出动画、滚动流畅 |

## 4. 验收标准

### 4.1 剪贴板监听

- [ ] 在任意 App 中复制文字，Ballet Clipboard 自动记录
- [ ] 在浏览器中复制链接，正确识别为 URL 类型
- [ ] 复制图片（截图、网页图片等），原图保存
- [ ] 连续复制相同内容不产生重复记录

### 4.2 菜单栏与面板

- [ ] 菜单栏显示粉色芭蕾图标
- [ ] 点击图标弹出 360×500 的历史面板
- [ ] 点击面板外区域自动关闭
- [ ] App 不出现在 Dock 栏

### 4.3 列表与卡片

- [ ] 最新复制内容排在最上面
- [ ] 文字卡片显示文字预览
- [ ] 图片卡片显示缩略图
- [ ] 链接卡片显示 URL 并带有特殊标识
- [ ] 每条记录显示相对时间（刚刚、X分钟前、X小时前…）

### 4.4 交互操作

- [ ] 点击文字卡片 → Cmd+V 可粘贴出文字
- [ ] 点击图片卡片 → Cmd+V 可粘贴出图片
- [ ] 点击链接卡片 → Cmd+V 可粘贴出链接
- [ ] 置顶卡片始终排在最上方，有金色标识
- [ ] 删除后卡片从列表移除，图片文件同步删除

### 4.5 搜索

- [ ] 输入关键词实时过滤
- [ ] 无匹配时显示友好提示

### 4.6 设置

- [ ] 保留期限 1/3/5 天可选
- [ ] 过期数据自动删除（含图片文件）

### 4.7 自启动

- [ ] 重启 Mac 后 App 自动启动

## 5. 非功能需求

| 类别 | 要求 |
|------|------|
| 性能 | CPU 占用 <1%，内存 <50MB |
| 存储 | 图片原图保存，数据库定期清理 |
| 兼容性 | macOS 14+ |
| 安全 | 本地存储，不上传任何数据 |
| 可用性 | 界面简洁，操作直觉，不需要教程 |
