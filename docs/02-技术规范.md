| 策略 | 适用场景 |
|------|----------|
| `@preconcurrency import CoreData` | Core Data 全局可变变量 (`NSMergeByPropertyObjectTrumpMergePolicy`) |

| `@MainActor` | AppDelegate 生命周期方法、ViewModel |

## 2. 架构：MVVM + Combine

```
┌─────────────────────────────────────────────┐
│  View Layer (SwiftUI)                       │
│  ContentView / ClipCardView / SearchBarView │
│      ↕ @EnvironmentObject                   │
│  ViewModel Layer                            │
│  ClipboardViewModel (@MainActor,            │
│    ObservableObject)                        │
│      ↕ Combine Publishers                   │
│  Service Layer                              │
│  ClipboardMonitor / PersistenceController   │
│  ImageStore / CleanupEngine                 │
│      ↕                                      │
│  Data Layer                                 │
│  Core Data (SQLite) + File System           │
└─────────────────────────────────────────────┘
```

### 各层职责

| 层 | 职责 | 不负责 |
|------|------|--------|
| View | 渲染 UI、接收用户交互 | 数据获取、业务逻辑 |
| ViewModel | 状态管理、搜索过滤、复制/删除操作、启动监听 | 剪贴板轮询、数据库 SQL |
| Service | 剪贴板轮询、类型识别、去重、持久化、图片 I/O | UI 展示 |
| Data | 数据存储与查询 (Core Data) | 业务逻辑 |

### 数据流

```
用户 ⌘C
  → NSPasteboard 更新
  → ClipboardMonitor (0.5s Timer) 检测 changeCount
  → 类型识别 (TIFF? → 图片 / string? → 文字/URL)
  → 去重检查 (文字比对 / 与最近一条比对)
  → performBackgroundTask 写入 Core Data
  → NSManagedObjectContextDidSave 通知
  → ViewModel.fetchItems() 刷新
  → @Published items 更新
  → SwiftUI 自动刷新列表

用户点击卡片
  → ViewModel.copyToClipboard()
  → NSPasteboard.clearContents() + setString/writeObjects
  → 更新 entry.timestamp (提至最前)
```

## 3. 数据模型

### Core Data Entity: ClipboardEntry

| 属性 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 (optional=NO) |
| contentType | String | "text" / "image" / "url" |
| textContent | String? | 文字内容 / URL 字符串 |
| imageFileName | String? | 图片文件名 (UUID.png) |
| timestamp | Date | 复制时间 |
| isPinned | Bool | 是否置顶 |

**索引**：
- `(isPinned, timestamp)` — 置顶优先 + 时间排序
- `(timestamp)` — 时间排序 + 清理查询

### 图片存储

```
~/Library/Application Support/BalletClipboard/Images/
├── {UUID}.png
└── ...
```

- 数据库只存文件名，ImageStore 负责路径管理
- 删除记录时同步删除对应图片文件
- 图片格式：PNG (从 NSPasteboard TIFF 转换)

## 4. 核心模块

### 4.1 ClipboardMonitor

```
start() → 记录 changeCount
  → Timer (0.5s, tolerance 0.1s)
  → checkPasteboard()
    → changeCount 变化？
    → 检查 pasteboard.types
    → .tiff / .png → readObjects(NSImage) → handleImage()
    → .string → handleText() → URLChecker.isURL()
  → 去重：与最近一条同类型记录比对
  → 非重复 → performBackgroundTask → save()
```

### 4.2 PersistenceController

```
单例 .shared
├── NSPersistentContainer(name: "BalletClipboard")
├── viewContext.automaticallyMergesChangesFromParent = true
├── viewContext.mergePolicy = NSMergeByPropertyObjectTrumpMergePolicy
└── performBackgroundTask { context in ... save() }
```

### 4.3 ClipboardViewModel

```
@MainActor + ObservableObject
├── @Published items: [ClipboardEntry]
├── @Published searchText: String
├── filteredItems: 搜索过滤计算属性
├── copyToClipboard() → NSPasteboard
├── togglePin() / delete() → Core Data save
├── fetchItems() → isPinned DESC, timestamp DESC, limit 200
└── 监听 NSManagedObjectContextDidSave → 自动刷新
```

## 5. 线程安全

| 操作 | 线程 | 机制 |
|------|------|------|
| 剪贴板读取 | 主线程 | NSPasteboard API 要求 |
| 数据写入 | 后台 | `performBackgroundTask` |
| UI 读取 | 主线程 | ViewModel @MainActor |
| 图片 I/O | 同步 (主线程调用) | ImageStore save/load |
| Core Data 通知 | 主线程 | `.receive(on: DispatchQueue.main)` |
