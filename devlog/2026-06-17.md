## 技术决策

| 决策 | 理由 |
|------|------|
| XcodeGen 替代手写 pbxproj | macOS 15 `plutil` 不支持旧式 plist 解析 |
| WindowGroup 调试模式 | 菜单栏 App 调试困难，先用窗口验证功能 |
| `generate_icons.py` 纯 Python 生成图标 | 无外部依赖，可随时重新生成 |
| `@unchecked Sendable` 策略 | 快速适配 Swift 6，后续逐步加强 |
| 列表上限 200 条 | 性能保护，实际够用 |
